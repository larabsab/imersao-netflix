## 📌 Sobre o Projeto
Este projeto foi desenvolvido durante a **Imersão Front-End com IA da Alura**. O objetivo principal foi recriar a interface icônica de seleção de perfis da Netflix, focando em **experiência do usuário (UX)** e **design responsivo**.

## ✨ Funcionalidades em Destaque
* **🌓 Alternância de Tema:** Sistema inteligente de Dark/Light Mode que salva sua preferência.
* **📱 Totalmente Responsivo:** Layout que se adapta perfeitamente do desktop ao celular.
* **🖼️ Avatares Personalizados:** Exibição fluida de imagens com carregamento otimizado (`lazy loading`).
* **🖱️ Efeitos Visuais:** Animações de borda e mudança de cor ao passar o mouse (hover).

## 🛠️ Tecnologias & Conceitos
* **HTML5:** Estrutura semântica para melhor acessibilidade.
* **CSS3:** Uso avançado de **Flexbox** e **Media Queries**.
* **JavaScript:** Manipulação do DOM e uso de `localStorage` para persistência de dados.
